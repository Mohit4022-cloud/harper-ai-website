Your Data Security is Our Top Priority

Harper AI is built with enterprise-grade security to protect your most sensitive sales data

Comprehensive Security at Every Level

Infrastructure Security

Our infrastructure is hosted on AWS with multiple availability zones, ensuring high availability and disaster recovery capabilities.

Data Protection

All data is encrypted at rest using AES-256 and in transit using TLS 1.3. We maintain strict data isolation between customers.

Access Control

Role-based access control, multi-factor authentication, and SSO support ensure only authorized users can access your data.

Security Features

Encryption

  • 256-bit AES encryption at rest
  • TLS 1.3 encryption in transit
  • Encrypted backups
  • Key management via AWS KMS

Authentication

  • Multi-factor authentication (MFA)
  • Single Sign-On (SSO) via SAML 2.0
  • OAuth 2.0 support
  • Password complexity requirements

Network Security

  • Web Application Firewall (WAF)
  • DDoS protection
  • IP allowlisting
  • Private network isolation

Monitoring & Logging

  • 24/7 security monitoring
  • Comprehensive audit logs
  • Anomaly detection
  • Real-time threat intelligence

Compliance & Certifications

Data Handling Practices

Data Ownership

You own your data. We act solely as a data processor, never selling or sharing your information with third parties.

Data Retention

Configurable retention policies let you control how long data is stored. Automatic purging ensures compliance with your policies.

Data Portability

Export your data anytime in standard formats. Full API access ensures you're never locked in.

Data Deletion

Request data deletion at any time. We ensure complete removal from all systems including backups within 30 days.

Security Operations

Incident Response

Our dedicated security team maintains a 24/7 incident response capability with defined escalation procedures and communication protocols. In the unlikely event of a security incident, affected customers are notified within 72 hours.

Vulnerability Management

Regular penetration testing, automated vulnerability scanning, and a responsible disclosure program help us identify and address potential security issues before they can be exploited.

Employee Security

All employees undergo background checks and security training. Access to production systems is limited to essential personnel with multi-factor authentication and audit logging.

Business Continuity

Comprehensive disaster recovery plans, regular backups, and geographically distributed infrastructure ensure your data remains available even in worst-case scenarios.

Security Architecture

API Layer

Data Layer

Infrastructure Layer

Network isolation, firewall rules, intrusion detection

Security FAQ

Where is my data stored?

Data is stored in AWS data centers in the US, EU, or Asia-Pacific based on your preference. All facilities are SOC 2 certified.

How often are backups performed?

Automated backups occur every 6 hours with point-in-time recovery available for the past 30 days.

Can I use my own encryption keys?

Enterprise customers can bring their own encryption keys (BYOK) for an additional layer of control.

Do you support on-premise deployment?

Yes, enterprise customers can opt for on-premise or private cloud deployment options.
